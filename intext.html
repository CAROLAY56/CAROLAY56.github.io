HTML

¿Qué es HTML?

HTML es el lenguaje con el que se define el contenido de las páginas web. A su vez, las tres siglas provienen de las iniciales de Lenguaje de Marcado de Hipertexto y, en esencia, es un leguaje de marcas que después lee el navegador. Con éste lenguaje podemos: definir el texto, las imágenes, listas, tablas, vídeos, etcétera que tendrán nuestras páginas web.

Estructura básica en HTML

Estructura

-!DOCTYPE html: Declara que el documento es HTML5.
-html: Etiqueta raíz que encierra todo el documento.
-head: Contiene metadatos sobre la página, como el título, codificación de caracteres, hojas de estilo CSS y scripts JavaScript.
-title: Define el título de la página, que aparece en la barra del navegador o en las pestañas.
-body: Contiene todo el contenido visible de la página web.

Etiquetas en HTML

¿Qué son las etiquetas HTML?

Las etiquetas HTML son comandos que le permiten al navegador saber cómo presentar el material. Como regla, las etiquetas se presentan en formato de apertura y cierre y están rodeada por varios atributos que complementan la información sobre el elemento..

Tips para usar HTML

Tips

-Asegúrate de incluir siempre la estructura básica de un documento HTML
-Utiliza etiquetas HTML semánticas como header, nav, main, article, y footer. Esto mejora la accesibilidad y la comprensión del contenido para otros desarrolladores y motores de búsqueda.
-Mantén tu código HTML bien indentado. Esto facilita la lectura y el mantenimiento del código. Usa espacios o tabulaciones de manera consistente.
-Asegúrate de cerrar todas las etiquetas HTML correctamente. Esto es crucial para evitar errores de visualización en los navegadores.
-Mantén el CSS y JavaScript en archivos externos en lugar de incluirlos directamente en el HTML. Esto hace que tu código sea más limpio y fácil de mantener.
-Usa comentarios en tu código HTML para explicar secciones o elementos específicos. Esto es útil para ti y para otros desarrolladores que puedan trabajar en el mismo código en el futuro.
